# Migration: v14 to v15

`v15` is a breaking release.

The main changes are:

- async APIs are cancellation-aware
- `IChatStorage.GetChatsAsync(...)` now returns `IAsyncEnumerable<ChatMetadata>`
- `ChatMetadata` is immutable
- storage providers use typed exceptions
- `IConnector.PostAsync(...)` and `ICommandHandler.HandleAsync(...)` now accept `CancellationToken ct = default`

The full migration guide lives in the repository root:

- [MIGRATION.md](../../MIGRATION.md)
