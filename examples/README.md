# Vermilion examples

- `QuickStart.Telegram.MongoDb` — MongoDB based chat storage.
- `QuickStart.Telegram.AzureTable` — Azure Table based chat storage.
- `QuickStart.Telegram.SqlServer` — SQL Server based chat storage.
